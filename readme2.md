# MAO_Biblioteca_Hidrolatos

Repositorio colaborativo de fichas técnicas de hidrolatos para aromaterapia, cosmética y fitoterapia, basado en literatura especializada y experiencia profesional. Este proyecto recopila y organiza información de referencia sobre plantas destiladas, usos, propiedades y composición, facilitando el acceso a datos claros y actualizados para la comunidad hispanohablante.

## Estructura del repositorio

- `Fichas_Plantas/`  
  Carpeta principal con archivos `.md` individuales, uno por especie/familia botánica.
- `knowledge_log.md`  
  Archivo de conocimiento vivo, pendientes, dudas y decisiones metodológicas.
- `README.md`  
  Este archivo introductorio.

## Contenido de cada ficha
Cada ficha incluye:
- Identificación botánica y taxonómica
- Descripción de la materia prima y origen
- Propiedades físico-químicas (pH, olor, color, solubilidad, conservación)
- Composición química clasificada (diferenciando AE/hidrolato)
- Propiedades terapéuticas (por sistema y uso)
- Indicaciones y aplicaciones (tópico, oral, ambiental, cosmética)
- Seguridad y contraindicaciones
- Compatibilidad aromática y bibliografía de referencia
- Notas sensoriales y aspectos diferenciales
- Referencias bibliográficas completas

## Cómo contribuir
- Revisa la plantilla base y ajusta tus fichas al formato estándar.
- Cita siempre la bibliografía utilizada o indica los vacíos o dudas en la sección de pendientes.
- Agrega tus aportes como pull request o issue para revisión colaborativa.
